# Hackspiration - Scholar Stream Project

## Team Collaboration Repository

This repository contains the implementation of the **Scholar Stream Smart Contract** built using Algorand AlgoKit and ARC4 smart contract framework.

---

## Project Description

Scholar Stream is a blockchain-based programmable money streaming vault that allows:

- Time-based fund release
- Delegated withdrawals
- Smart contract controlled disbursement
- Secure and transparent streaming payments

---

## Tech Stack

- Algorand Blockchain
- AlgoKit
- ARC4 Smart Contracts
- Python (AlgoPy)
- Docker LocalNet
